Sobre mim

Onde Engenharia
encontra o Código

Sou Felipe Ribeiro, desenvolvedor Full Stack cursando Desenvolvimento de Software na Fatec-Jacareí e formado em Engenharia de Produção pela UNIVESP. Essa combinação me deu uma visão única: penso tanto nos processos que fazem um sistema funcionar quanto na experiência de quem vai usá-lo.

Ao longo do curso, acumulei experiência prática com TypeScript, JavaScript, React e Node.js, passando pelo desenvolvimento de interfaces responsivas até APIs REST e banco de dados — tanto relacionais quanto não-relacionais (MongoDB). No 4° semestre já trabalhava com Desenvolvimento Web III e técnicas avançadas de modelagem.

Como membro ativo da Capydev-jac, participei de projetos colaborativos reais — com versionamento, pull requests, code review e entregas com demo em vídeo. Tenho conquistas GitHub de Pull Shark e Quickdraw, reflexo do ritmo de contribuição.
